# Sort 实验 — 冒泡 / 归并 / 快速排序对比

## 算法

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
|------|-----------|-----------|--------|
| 冒泡排序 | O(n²) | O(1) | 稳定 |
| 归并排序 | O(n log n) | O(n) | 稳定 |
| 快速排序 | O(n log n) 平均 / O(n²) 最坏 | O(log n) | 不稳定 |

快速排序使用**随机化基准**避免最坏情况。

## 编译

```bash
g++ -O2 -std=c++11 sort.cpp -o sort
./sort
```

## 任务说明

| 任务 | 目的 | n 范围 | 数据集 |
|------|------|--------|--------|
| Task 1 | 固定规模，多种子稳定性 | n = 10000 | seed=12345 / 67890 |
| Task 2 | 规模扩展性测试 | n = 10, 100, 1000, 10000, 50000 | 随机整数 |
| Task 3 | 不同输入分布 | n = 10000 | 顺序 / 逆序 / 随机 |

## 输出

- 终端打印每个算法在每个 (n, seed, 分布) 下的运行时间（ms）
- 同时把样本数据写入 `data/sort_*.csv`
