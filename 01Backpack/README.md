# 0-1 背包实验 — 四种算法对比

## 算法

| 算法 | 时间复杂度 | 空间复杂度 | 适用规模 |
|------|-----------|-----------|----------|
| 暴力枚举 | O(2ⁿ) | O(n) | n ≤ 24 |
| 动态规划（滚动数组） | O(nC) | O(C) | 任意 |
| 贪心（价值/重量比） | O(n²) 或 O(n log n) | O(n) | 任意（结果不一定最优） |
| 回溯 + 上界剪枝 | < O(2ⁿ) 实际很快 | O(n) | n ≤ 60 |

## 编译

```bash
g++ -O2 -std=c++11 01backpack.cpp -o 01backpack
./01backpack
```

## 实验设计

### 小规模（n ≤ 24）
四种算法**全部运行**，用于验证正确性和时间复杂度趋势。
- n = 10, 15, 20, 24
- C = 10000, 100000, 1000000
- 共 4 × 3 = 12 组

### 大规模（n ≥ 1000）
**只跑 DP + 贪心**（暴力/回溯不可行）。
- n = 1000, 2000, 3000, 4000, 5000, 6000, 7000, 8000, 9000, 10000, 20000
- C = 10000, 100000, 1000000
- 共 11 × 3 = 33 组
- C = 1000000 且 n ≥ 8000 时 DP 也可能超过 10s，本实验省略该段

## 物品生成

- 重量 wᵢ ∈ [1, 100] 整数
- 价值 vᵢ ∈ [1.0, 1000.0] 浮点

种子固定，每次运行结果可复现。

## 输出

- 终端打印每个 (n, C) 组合下各算法的**最优解**和**耗时（ms）**
- 同时把物品样本写入 `data/ks_*.csv`
  - n ≤ 100：写入全部物品
  - n > 100：写入前 20 + 中间 30 + 末 20（避免大文件）

## 文件

- `01backpack.cpp` — 主程序
- `data/` — 生成的物品样本 CSV
